# README
Pickitup! is application that let you register the pickups of coins.

## Installation
Pickitup! is based on:
 * Ruby 1.9
 * Ruby on Rails 4.1

## Change-log

### Proposed features
 * Notifications
 * Filter pickups
 * Location as City+Country
 * Enforce auth
 * Received option in pickups
 * Edit location from coordinates
 * Yearly stats
